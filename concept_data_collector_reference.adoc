= 공급업체별 참조 자료


[role="lead"]
이 섹션의 항목에서는 공급업체별 참조 정보를 제공합니다. 대부분의 경우 데이터 수집기 구성은 간단합니다. 경우에 따라 데이터 수집기를 올바르게 구성하기 위해 추가 정보나 명령이 필요할 수 있습니다.

데이터 수집기에 대한 정보를 보려면 왼쪽의 메뉴에서 * vendor * 를 클릭하십시오.
